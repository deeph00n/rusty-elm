# rusty-elm
## Setup (about 3min without IntelliJ IDEA)
- clone repo
- install [rust](https://win.rustup.rs/)
- install [elm](https://github.com/elm/compiler/releases/download/0.19.1/installer-for-windows.exe)
- (optional) install [IntelliJ IDEA](https://www.jetbrains.com/idea/download/download-thanks.html?platform=windows) with
    - [rust plugin](https://intellij-rust.github.io/)
    - [elm plugin](https://github.com/klazuka/intellij-elm#install)
- open PowerShell in repo directory
- run `choco install psake`
- run `psake`
